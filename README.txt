Quake II Multiplayer PVE

* Build instructions

	- Compile game.sdf and put the resulting gamex86 in your quake2 baseq2 folder

	- Alternatively, just use the gamex86 here and put it in your baseq2 folder

* Gameplay instructions

	- Start a deathmatch multiplayer game

	- Pick up a weapon to choose class
 		-machinegun == Tank 

		-shotgun == Medic

		-grenadelauncher == DPS

	- Kill all of the enemies in each wave to advance to the next

	- 1 of 3 Powerups is given at the end of each round (effects are class dependent)

	- Survive for 4 rounds to win the game.
